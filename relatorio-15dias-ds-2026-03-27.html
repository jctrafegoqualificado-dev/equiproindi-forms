Configurar rastreamento de conversões no Meta

Meta Ads
2
Criar campanha de conversão/leads no Meta
A campanha atual é de reconhecimento. Criar uma segunda campanha com objetivo de Lead Generation ou Mensagens para WhatsApp, aproveitando o alcance gerado pelo awareness.
Meta Ads
3
Revisar rastreamento Google — WhatsApp e formulário
Com 745 cliques e 9 conversões, provavelmente existem mais contatos que não estão sendo contabilizados. Configurar conversão por clique no número e envio de formulário no site.
Google Ads
4
Criar campanha de remarketing com portfólio visual
Usuários que visitaram o site mas não converteram são o público mais quente. Remarketing com imagens dos projetos de interiores no Meta + Display Google reativa esse interesse.
Meta + Google
5
Restaurar acesso à API do Google Ads
A API do Google Ads está retornando erro de acesso. Verificar se o projeto no Google Cloud tem a API "Google Ads API" habilitada e se o developer token está aprovado.
Google Ads

Avaliar reativação da campanha "De Styll"
Segunda campanha pausada no Google. Analisar motivo da pausa e considerar reativação com palavras-chave complementares à campanha principal, aumentando cobertura de busca.
